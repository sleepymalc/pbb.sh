## Problem

Machine unlearning (MU) — the task of selectively removing the influence of specific training data from a deployed model — has matured rapidly, yet *evaluating* unlearning algorithms remains a long-standing open problem. The dominant evaluation paradigm relies on membership inference attacks (MIAs): measure how well an adversary can detect whether a data point was used during training, and treat that attack success rate as a proxy for removal quality. This approach is fundamentally brittle. Crucially, retraining from scratch is the gold-standard for data removal, yet it is not guaranteed to minimise MIA success compared to approximate methods — meaning the theoretically optimal algorithm can *appear worse* under the very metric meant to assess it. Results also shift with the choice of attack, dataset composition, and evaluation protocol, with no unified theoretical framework in sight.

## Key Idea

We introduce the **unlearning sample inference game**, a cryptographic-style two-player framework. The data is split into three disjoint parts: a *retain* set $\mathcal{R}$, a *forget* set $\mathcal{F}$, and a *test* set $\mathcal{T}$. The defender (challenger) trains a model on $\mathcal{R} \cup \mathcal{F}$ and then unlearns $\mathcal{F}$. The adversary is given samples drawn from *either* $\mathcal{F}$ or $\mathcal{T}$ — but not told which — and must guess the source. Intuitively, if the unlearned model has genuinely forgotten $\mathcal{F}$, the adversary gains no advantage from querying it; the forget-set samples look no different from the never-seen test-set samples.

## Approach

Using the notion of **advantage** — the difference in the adversary's success rate across the two scenarios — we define **Unlearning Quality** $\mathcal{Q} = 1 - \sup_{\mathcal{A}} \operatorname{Adv}(\mathcal{A}, \text{Ul})$, where the supremum ranges over all adversaries. Higher $\mathcal{Q}$ means better unlearning. The framework comes with two provable guarantees: (1) for the gold-standard retraining method, $\mathcal{Q} = 1$ always holds; (2) for any $(\epsilon, \delta)$-certified removal method,[^1] $\mathcal{Q} \geq \frac{4 - 4\delta}{e^{\epsilon} + 1} - 1$. To our knowledge, this is the first unlearning evaluation metric with formal guarantees of either kind.

Computing $\mathcal{Q}$ exactly requires enumerating an exponential number of dataset splits, which is infeasible in practice. We therefore propose an efficient approximation — the **SWAP test** — which selects one random split, computes the advantage, then swaps $\mathcal{F}$ and $\mathcal{T}$ and averages the two results. This two-split procedure provably preserves the zero-grounding property ($\mathcal{Q} = 1$ for retraining) while reducing computation to a practical level.

[^1]: [https://arxiv.org/abs/1911.03030](https://arxiv.org/abs/1911.03030)

## Result

Empirically, $\mathcal{Q}$ produces consistent, stable rankings of unlearning algorithms across varying dataset sizes and unlearning-portion parameters $\alpha$, where existing MIA-based metrics fluctuate. When evaluated against differentially private baselines with known $(\epsilon, \delta)$ budgets, $\mathcal{Q}$ exhibits the theoretically predicted negative correlation with $\epsilon$, directly validating the certified-removal lower bound. Standard MIA AUC and prior composite scores fail both tests. This makes $\mathcal{Q}$ the first empirically demonstrated unlearning metric that is simultaneously grounded in theory and practical to compute.
