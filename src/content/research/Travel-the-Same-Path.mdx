## Overview

Inspired by [*Exact Combinatorial Optimization with Graph Convolutional Neural Networks*](https://arxiv.org/abs/1906.01629), this project explores using **imitation learning** to speed up exact solvers for the [traveling salesman problem (TSP)](https://en.wikipedia.org/wiki/Travelling_salesman_problem). The key insight is that *accelerating only the decision-making steps of an exact algorithm can preserve exactness* while significantly reducing solve time.

## Approach

We formulate TSP as an integer linear program and solve it via **branch and bound**. Since branching involves a sequence of variable-selection decisions, we model it as a Markov Decision Process and train a Graph Convolutional Neural Network to imitate the branching strategy of SCIP. The model learns to predict which variable to branch on at each step, and the methodology generalizes to any exact algorithm that involves sequential decision-making (e.g., cutting planes).
